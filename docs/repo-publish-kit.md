## 7. What To Avoid In Public Copy

Avoid phrases like:

- `production-ready`
- `enterprise-grade autonomous system`
- `fully complete runtime`
- `official Feishu runtime`

These overclaim the current Bundle A reality.

---

## 8. Recommendation

If you want to move fast, use:

- the recommended GitHub description
- the first public release summary
- the short Chinese post as the primary market-facing announcement

That is the safest messaging package for the current readiness level.
